# 📊 Sales Performance Dashboard (Power BI)

## 🔍 Overview

This project analyzes sales, profit, and customer trends using Power BI.

## 📈 Key Features

* KPI metrics (Sales, Profit, Orders, Profit Margin)
* Sales trend analysis
* Region-wise performance
* Category-wise sales vs profit
* Top 5 products analysis
* Interactive filters (Year slicer)

## 🛠 Tools Used

* Power BI
* Excel

## 📊 Insights

* Identified top-performing regions and products
* Compared sales vs profitability across categories
* Analyzed trends over time

## 📁 Files Included

* Power BI Dashboard (.pbix)
* Dataset (.xlsx)
